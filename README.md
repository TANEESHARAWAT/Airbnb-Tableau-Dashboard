# Airbnb Data Analysis Dashboard using Tableau

## Project Overview
This project is an interactive Tableau dashboard created to analyze Airbnb listing data and discover meaningful insights about pricing, revenue trends, bedroom distribution, and location-based patterns.

The dashboard converts raw Airbnb data into easy-to-understand visualizations that help analyze market trends and support data-driven decision-making.

## Dashboard Preview

![Airbnb Dashboard](Dashboard.png)

## Features
- 📊 Average Price Analysis by Bedroom
- 📍 Price Analysis by Zipcode
- 🗺️ Geographic Price Visualization
- 📈 Revenue Trend Analysis Over Time
- 🏠 Bedroom Count Distribution
- 🔎 Interactive filters for better exploration

## Tools & Technologies
- Tableau Public
- Data Visualization
- Data Analytics
- Business Intelligence

## Key Insights
- Higher bedroom counts generally have higher average prices.
- Airbnb prices vary significantly across different zip codes.
- Revenue trends show changes in performance over time.
- Location plays an important role in Airbnb pricing.
